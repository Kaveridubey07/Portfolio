# 🌐 Personal Portfolio Website

A modern and responsive Personal Portfolio Website built using HTML and CSS. This project showcases personal information, skills, projects, and achievements through a clean and professional user interface.

## 🚀 Live Demo

🔗 https://portfolio-fb4o.onrender.com

---

### 📌 Project Overview

The Personal Portfolio Website is designed to establish a professional online presence and showcase personal achievements, technical skills, projects, and career objectives.

The website follows a clean and responsive design approach, making it accessible across desktop, tablet, and mobile devices. It serves as a digital resume and portfolio for recruiters, hiring managers, and potential collaborators.
